(route) Add a new roommate to a room

Route:
MethodPath
POST room/add-roommate
Body Parameters:
Name Type Description
existing_roommate String The name of the already existing roommate
new_roommate String The name of the roommate to be added newly to the room
room_nm String The name of the new room the new roommate is getting added to
notification_id String The notification id linked to the room join request.
Response:
Name Type Description
message String See description of the different status codes
Response Code:
Type Description
200 New Roommate successfully added
404 Room not found
404 Room not found. Create or Join a room
404 User not found
404 New roommate not found
404 Old roommate not found
500 Error message from backend
